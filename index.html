What is Polka?

Polka extends the famous mootools-more accordion with some new features.

Tell me more what features could i expect?

  • Open Accordion items via the url
  • Url Hash Updating
  • Optional build in Aria Support for more accesibillity
  • If the Accordion wrapping element dosn`t have an id, polka will create one
  • Extends the mootools-more accordion
  • Some new callbacks

Quickstart

Options

anchorItem
Polka expects a query selector to detect the wrapper of each accordion. Default '.accordion-wrap'
jumpToOnClick
Polka automaticly scrolls to the beginning of the active toggler element top. Default true
setAriaSupport
Adding some features for better accessibility. Default true
ariaTabListElement
If setAriaSupport, you should define wich element should get the tablist Attribute. Default '.accordion-wrap'
autoItemName
If Polka didn`t detect an id, wich is nessecery for a anchor jump, polka will automaticcly create this id`s. With this option you can define a own default id name. Default 'accordion-item'
scrollOffset
Maybe you need some offset for the scrolling. Default { x: 0, y: 0 }
