Tremolo - A Mootools Accordion Extension
by Joe Ray Gregory

Tremolo is an Mootools Accordion Extension. The basic Mootools Accordion is great but sometimes you need more.

Tremolo extends the Accordion with the following features:

Url Extension
Adds url support to each element. Onclick every element set an anchor fragment to the url. So it` posible to auto open an element. Tremolo has an internal id checker, so if an element dosn` have an id tremolo autgenerates one.
Smooth Scroll
If you link to an accordion fragment tremolo srolls automaticly to target via Mootools smooth scroll
Keyboard controll
Controll the Accordions via your keyboard. tab and arrow keys are supported.
